# stockWatch
Stock Market price tracker and analyzer

A full-stack stock tracking web application built with FastAPI and React.

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- MySQL
- JWT Authentication

### Frontend
- React
- Chart.js
- Axios

## Features

- User Authentication
- Stock Search
- Stock Details
- Historical Price Charts
- Custom Watchlists
